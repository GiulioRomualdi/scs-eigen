Function documentation

void ScsEigen::Logger::setLevel(Level level)

set the level of the verbosity.

Note

If the level is set to Logger::Level::error only the errors will be print. If set to Logger::Level::info warnings and errors are printed. Set the level equal to Logger::Level::none if you want to disable all the messages.
